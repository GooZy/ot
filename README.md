# OT

Okr tracker

## V1.0

### 功能

- 选择双月新建目标
  - O-K-R结构
- 目标值
  - 具体的目标，比如教学视频要看30集，那目标值就是30
  - 默认值为10，代表通常意义上的10分
- 权重
  - 每一个K的权重，总和为1
- 按年份具有折叠功能
- 评论功能
  - 给自己本次双月的成果做个总结
- 本地存储
  - 或者搭建方便，每次启动数据不丢失

原型：

月份：2020年1月~2020年2月

| O    | K    | R    | 目标值 | 权重 |
| ---- | ---- | ---- | ------ | ---- |
| O1   | K1   | 2    | 10     | 30%  |
|      | K2   | 1    | 30     | 30%  |
| O2   | K3   | 0    | 10     | 20%  |
| O3   | K4   | 4    | 20     | 20%  |

总计：`K1(2/10)*30%+K2(1/30)*30+K3(0/10)*20%+K4(4/20)*20%`
